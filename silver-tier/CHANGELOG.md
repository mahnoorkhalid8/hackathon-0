# Changelog

All notable changes to the Silver Tier Digital FTE project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-13

### Added
- Initial release of Silver Tier Digital FTE
- Core orchestrator with event queue management
- Context loader for memory vault access
- Reasoning engine with iterative analysis
- Task router with approval rule evaluation
- State manager for dashboard updates
- Executor for skill-based task execution
- File watcher for file system monitoring
- Time watcher for scheduled tasks
- MCP client for external integrations
- Memory vault with markdown-based storage
- Dashboard.md for system status
- Company_Handbook.md for policies
- Plan.md for reasoning transparency
- Three example skills: email_responder, report_generator, data_analyzer
- Comprehensive configuration system
- Approval workflow (Needs_Action vs Needs_Approval)
- Complete documentation (README, QUICKSTART, ARCHITECTURE)
- Demo script for testing
- Logging infrastructure
- .gitignore for version control
- MIT License

### Features
- Local-first architecture
- Multi-watcher trigger system
- Human-in-the-loop approval
- Skill-based execution
- MCP integration capability
- Complete audit trail
- Safety-first defaults
- Extensible design

### Documentation
- README.md - Project overview
- QUICKSTART.md - Getting started guide
- ARCHITECTURE.md - Technical design document
- PROJECT_SUMMARY.md - Complete project summary
- Inline code documentation

### Configuration
- fte_config.yaml - Main system configuration
- approval_rules.yaml - Approval policies
- schedule_config.yaml - Scheduled tasks
- mcp_config.yaml - MCP server registry
- watcher_config.yaml - Watcher settings
